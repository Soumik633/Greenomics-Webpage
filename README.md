# 🌿 Greenomics – Deforestation & Its Economic Impact

Greenomics is an interactive web project designed to raise awareness about **deforestation and its economic, environmental, and social impacts**.  
The website presents educational content, live statistics, data visualizations, and policy information to help users understand the scale of forest loss and the importance of conservation.

---

## 🌍 Project Overview

Every year the world loses **millions of hectares of forests**, impacting biodiversity, climate stability, and economic systems.  
This project aims to:

- Educate users about deforestation
- Show its economic consequences
- Present real statistics and data visualizations
- Promote awareness and sustainable actions

The project focuses especially on **India's forest landscape and deforestation trends**.

---

## ✨ Features

### 🌳 Interactive Website Sections
- Hero section introducing deforestation crisis
- Inspirational story of **Jadav "Molai" Payeng – The Forest Man of India**
- Explanation of deforestation and global statistics
- Effects categorized into:
  - Economic
  - Environmental
  - Social
  - Health

### 🗺️ India Deforestation Map
- Visual map showing **major deforestation zones in India**
- Forest density classification
- State-wise forest cover table

### 📊 Data Visualization
Charts built using **Chart.js**:
- Top states with forest loss
- Forest cover distribution

### ⏱️ Live Environmental Counters
Real-time style statistics showing:
- Trees lost
- Trees planted
- CO₂ released
- Hectares of forest lost

### 📋 Government Schemes
Information about initiatives like:
- Green India Mission
- CAMPA
- National Afforestation Programme
- REDD+
- Project Tiger

### 🌱 Awareness Section
Practical actions users can take to reduce deforestation impact.

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website
- **CSS3** – Styling and responsive layout
- **JavaScript (Vanilla JS)** – Interactivity and data logic
- **Chart.js** – Data visualization
- **html2canvas & jsPDF** – Certificate and export features
- **QR Code JS** – QR code generation
